	**MISCELLANEOUS GIT COMMANDS AND THEIR FUNCTIONS:** ================================



			**git init** --> initialize new Git repo locally

&nbsp;	

&nbsp;			**git clone <url>** --> Copy existing remote repo locally



&nbsp;			**git remote add origin <url>** --> Link local repo to GitHub



&nbsp;			**git branch -M main** --> Ensure branch is named main



&nbsp;			**git status** --> Check changes



			**git add <file>** --> Stage single file



			**git add .** --> Stage all changes



			**git commit -m "Changes made here"**  --> Commit staged changes



			**git commit --amend -m "New message"** --> Edit last commit



&nbsp;			**git push -u origin main** --> First push



			**git push** --> Push future commits



			**git pull --rebase** --> Update local branch with remote changes
